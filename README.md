## Installation  

1. Clone the repository:  
   ```sh
   git clone https://github.com/Margarita23/StarTodoList.git  
   cd StarTodoList
   ```

2. Install dependencies:  
   ```sh
   npm install 
   ```

3. Run the application:  
   ```sh
   npx expo start 
   ```

   If using an emulator or device:  
   ```sh
   npm run android  # for Android  
   npm run ios      # for iOS  
   ```
